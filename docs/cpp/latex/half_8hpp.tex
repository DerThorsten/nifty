Default rounding mode. This specifies the rounding mode used for all conversions between \hyperlink{classhalf__float_1_1half}{half}s and {\ttfamily float}s as well as for the \hyperlink{namespacehalf__float_1_1detail_aa886fe2fe51f0ead5eca42060889b331}{half\+\_\+cast()} if not specifying a rounding mode explicitly. It can be redefined (before including \hyperlink{half_8hpp}{half.\+hpp}) to one of the standard rounding modes using their respective constants or the equivalent values of {\ttfamily std\+::float\+\_\+round\+\_\+style}\+:

\subsubsection{\texorpdfstring{H\+A\+L\+F\+\_\+\+R\+O\+U\+N\+D\+\_\+\+T\+I\+E\+S\+\_\+\+T\+O\+\_\+\+E\+V\+EN}{HALF\_ROUND\_TIES\_TO\_EVEN}}
{\footnotesize\ttfamily \#define H\+A\+L\+F\+\_\+\+R\+O\+U\+N\+D\+\_\+\+T\+I\+E\+S\+\_\+\+T\+O\+\_\+\+E\+V\+EN~0}

Tie-\/breaking behaviour for round to nearest. This specifies if ties in round to nearest should be resolved by rounding to the nearest even value. By default this is defined to {\ttfamily 0} resulting in the faster but slightly more biased behaviour of rounding away from zero in half-\/way cases (and thus equal to the \hyperlink{namespacehalf__float_1_1detail_ab5b61d50f0c874faf8d0a949386242ff}{round()} function), but can be redefined to {\ttfamily 1} (before including \hyperlink{half_8hpp}{half.\+hpp}) if more I\+E\+E\+E-\/conformant behaviour is needed. \mbox{\Hypertarget{half_8hpp_abdcd774b81d03564edec8523484d02bf}\label{half_8hpp_abdcd774b81d03564edec8523484d02bf}}
